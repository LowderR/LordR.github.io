Project Description

In this project, I downloaded college basketball tournament data from Kaggle,
which contains cleaned data of previous season's results, team information, tournament introductions,
and some additional data that might be useful in predicting the tournament results for the 2022 season.

Constructing Dataset

After I downloaded the data, I first did some preliminary research about the tournament rules, how success is measured, and in general, how the team performance is measured. Then, I went through all the files and read the description provided by Kaggle to find out the specific meaning of each variable. Third, based on my understanding of the tournament and the data I have, choose the prediction metric I would like to use in my prediction.

First, I chose each team's average score per game and the winning probability for each team in each playing season as my primary prediction measure.
Second, I chose several variables directly related to how initiative the team is in each game from the result detail file provided by Kaggle. I used these variables to measure team efficiency and included them as my prediction feature.
Third, I also think team rank during the playing season is also a helpful feature in terms of predicting game results. I also included some measures of team ranking, including mean, median, and whether the team has ranked up from the first day of the game to the last day in my predicting features.
After extracting all the features I needed, I constructed a training and testing dataset from the past year's result file and merged my feature into the dataset.

Building Model

Since it is a classification question, I chose to build two models for the prediction and compare the model accuracy, choosing the best one to predict the 2022 result.
The first model I chose is logistic regression with l2 penalty. I chose to use l2 penalty even when I don't have too many predictors because I suspect there might be a multicollinearity issue in my model, and I'd like to avoid such impact.
The second model I choose is a stochastic gradient with early stopping and the loss function after using my training data to train my models and take them to my testing data. The testing accuracy measured by the log loss function is slightly higher, which is about 0.613; so, I chose this model to predict the 2022 results.

Reflection

I chose to join this Kaggle competition and work on this dataset is not because I'm such a basketball fan. On the contrary, my knowledge of basketball is minimal, and I'd like to train myself to work with something I'm not familiar with since it might be most of the case in the actual workplace. I want to get acquainted with this process, from doing background research to choosing the metric to use and building my own dataset using my data. Finally, select the possible model to deal with the problem.
During this process, it is the first part that I found most challenging; I need to take more time to understand this problem before moving on to the next part. Working on this project actually reminds me of what I need to keep improving next.
Also, about this question itself, due to my limited knowledge about basketball and the time constraint, I couldn't analyze each variable in the file carefully, which means that my model might leave out some valuable features. Also, I just started learning another tool, which is CVM, to make a classification. In the future, I would like to add additional features which I have not used here and try out other models to improve my predicting accuracy.
